#### 基本流程

1.在目录中新建组件文件，文件为js或jsx格式

2.引入reacet进行声明，将声明的组件进行暴露

3.在需要使用的文件中引入组件，而后进行使用



**声明组件：**

```js
import React from "react";

class Test extends React.Component {
	render() {
		return(
			<div className="test-div" onClick={() => handleClick()}>我是测试组件</div>
		)
	}
}

function handleClick() {
	console.log('点击咯！')
}

export default Test;
```



**使用组件：**

```js
import Test from './test.jsx'

function App() {
	return(
  	<div className="app-div">
    	<div>我是父组件<div/>
      <Test />
    </div>
  )
}
      
export default App;
```



**界面渲染**

```js
import React from 'react';
import ReactDOM from 'react-dom/client';
import App from './App.js';

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
	<React.StrictMode>
  	<App />
  </React.StrictMode>
);

```
