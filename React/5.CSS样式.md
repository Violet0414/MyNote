#### CSS样式书写

**1.行内样式**

```jsx
class App extends React.Component() {
	// ...
  render() {
  	return(
    	<div>
        <div style={{ background:'#ffffff', width:'200px', height:'200px', fontSize:'16px' }}>行内样式</div>
      </div>
    )
  }
}
```



**2.声明样式**

```jsx
class App extends React.Component() {
  const styles={    
    background:'#ffffff',
    width:'200px',
    height:'200px',
  }

	render() {
  	return(
    	<div>
      	<div style={styles}></div>
      </div>
    )
  }
}

```



**3.引入样式**

​	在开发过程中往往使用引用样式的方式书写CSS样式，同时存在一定的开发规范。

​	局部样式命名：**xxx.moudle.css**

​	全局样式命名：**xxx.css**

```jsx
import React from 'react'
import '../App.css'
import app from './app.css'

class App extends from React.Component() {
	render() {
  	return(
    	<div>
    		<div className={`${appStyle}` `${app.style}`}>混用引入样式</div>
      </div>
    )
  }
}
```
